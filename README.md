# React Calculator

A simple calculator built with React 18 and Vite.

![Screenshot](screenshot.png)

## Tech Stack

- React 18
- Vite 6
- CSS3

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Features

- Basic arithmetic operations (+, -, *, /)
- Decimal support
- Clear function
- Safe expression evaluation
